# botium-toys-internal-security-audit
This repository presents an internal cybersecurity audit for a fictional company, Botium Toys, using the NIST Cybersecurity Framework. It assesses security controls, compliance gaps (PCI DSS, GDPR, SOC), and risk exposure, and provides actionable recommendations to improve security posture and business continuity.

# Skills & Frameworks Demonstrated
- Internal security auditing
  
- Risk assessment and asset identification
  
- NIST Cybersecurity Framework (Identify function)
  
- Controls assessment (administrative, technical, physical)
  
- PCI DSS, GDPR, and SOC compliance analysis
  
- Security gap analysis and remediation planning
  
- Governance, Risk, and Compliance (GRC) fundamentals
